# Distribution and legal boundaries

This repository distributes original compatibility code, source-level patches,
documentation, and relay test code only.

It does not distribute:

- Minecraft or Legacy Console Edition source code as a complete source tree
- game executables or libraries
- Xbox 360 XEX files
- PlayStation PKG, SELF, or EBOOT files
- game ISOs, decrypted discs, title updates, or resource archives
- Xbox, PlayStation, Microsoft, Sony, or Nintendo SDKs
- firmware, keys, certificates, credentials, or signing material
- license-check removal or trial-to-full conversion patches
- Mojang, Microsoft, 4J Studios, or platform artwork/audio

Users are responsible for obtaining source, game files, firmware, and
toolchains lawfully and for following the licenses and laws that apply in their
jurisdiction. Do not upload locally built game artifacts to this repository or
attach them to GitHub releases.

not grant rights to third-party game code, assets, trademarks, SDKs, or build
outputs.
