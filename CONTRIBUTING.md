# Contributing

Keep contributions patch-only and reproducible.

- Do not commit game binaries, complete proprietary source trees, SDK files,
  firmware, keys, dumps, resource archives, or copyrighted game assets.
- Keep all packet changes behind `CONSOLE_LEGACY_RELAY` unless they are proven
  safe for the original platform network paths.
- Change the build ID when packet serialization or network behavior changes.
- Include a minimal reproduction, all client build hashes, and relay logs for
  network bugs.
- Add an automated relay test for routing or framing changes.
- Do not add branding, launcher features, unrelated gameplay mods, or license
  bypasses to this repository.
